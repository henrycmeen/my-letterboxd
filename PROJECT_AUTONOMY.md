# Project autonomy

Codex may find bounded bugs, implement and test issue-scoped work, push branches, open pull requests, and merge after green checks and independent review. Preserve the VHS-floor design contract in `AGENTS.md`.

Production deployment, real account/data changes, secrets, destructive operations, broad visual direction, or URL/base-path migration require Henry. Use Terra high for ambiguous multi-file UI behavior and Sol high for security, architecture, migrations, or independent final review. Model strength never expands A3.
