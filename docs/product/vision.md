# Product Vision

## Core Idea

Build a playful VHS-style "film floor" where movie discovery, ranking, and collecting feels tactile.

## Experience Principles

- Minimal UI chrome: the floor and cassettes should do most of the communication.
- Direct manipulation first: drag/drop interactions should feel physical and responsive.
- Fast idea-to-publish loop: new experiments should be easy to test and ship.

## Long-Term Direction

- Multiple cassette/case styles that can be swapped per project/theme.
- Reusable rendering pipeline for front/spine/back variants.
- Better curation flows for watchlists, shelves, and quick comparisons.
